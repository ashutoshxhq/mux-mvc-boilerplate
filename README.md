# simple-go-server
A simple go server setup using mux 

## How to use?
1. **Clone the Repo**
```
$ git clone https://github.com/idkashutosh/simple-go-server.git
```

2. **Change to Directory**
```
$ cd simple-go-server
```

3. **Get the Packages**
```
$ go get ./...
```

4. **Run the Project**
```
$ go run main.go
```

**Now open you browser at http://localhost:8080 to see the result**
